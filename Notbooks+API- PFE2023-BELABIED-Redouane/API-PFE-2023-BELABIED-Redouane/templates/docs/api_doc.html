Welcome to API Documentation

This documentation provides details about the API endpoints and their usage.

/api/v1/predict2

Method: POST

Description: Predicts the label of the uploaded image using a deep learning CNN model.

Request:

  • Form Data: my_image (image file)
  • Form Data: Authorization (API Token)

Response:

{
  "prediction": "predicted_label",
  "with accuracy": "Accuracy"
}

Note: The prediction is made using a pre-trained deep learning CNN model that has been trained on a large dataset. The predicted label belongs to one of the following classes: 'MildDemented', 'ModerateDemented', 'NonDemented', 'VeryMildDemented'.

/api/v1/auth

Method: POST

Description: Authentification.

Request:

  • Json Data: username (string)
  • Json Data: password (string)

Response:

{
  "access_token": "api Token"
}

Note: You need to be registred inorder to use this endpoint.
